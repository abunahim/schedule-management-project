# 🗓️ Schedule Management Project

A DevOps-style Schedule Management REST API built with **Java Spring Boot**, following a phase-by-phase build approach covering full-stack development, containerization, CI/CD, Kubernetes, caching, and frontend.

---

## 🛠️ Tech Stack

| Layer | Technology |
|---|---|
| Backend | Java 17, Spring Boot 3.5 |
| Database | PostgreSQL |
| Caching | Redis |
| Containerization | Docker |
| Orchestration | Kubernetes (kind) |
| CI/CD | GitHub Actions |
| Frontend | React + Nginx |
